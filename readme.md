# Udruga Mladih Santa Fe

Web stranica Udruge Mladih Santa Fe iz Sutomišćice koja prikazuje događanja i informacije o udruzi.

## Live demo
https://udrugaumsf.netlify.app

## Stranice

| Stranica | Opis |
|----------|------|
| index.html | Naslovnica |
| dogadjanja.html | Događanja |
| onama.html | O nama |
| kontakt.html | Kontakt |

## Funkcionalnosti

- **Navigacija** — responzivni izbornik
- **Događanja** — prikaz nadolazećih događaja
- **Kontakt forma** — obrazac za kontakt
- **Responzivni dizajn** — prilagođen mobilnim uređajima

## Tehnologije

- HTML5, CSS3 (Flexbox, CSS varijable)
- JavaScript (vanilla)
- Google Analytics 4
- Netlify hosting
